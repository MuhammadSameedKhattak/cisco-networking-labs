# Cisco Networking Labs

Hands-on Cisco Packet Tracer labs completed as part of CCNA-track coursework and internship practice at Dev Dabs, Islamabad.

## Labs Included

### 1. OSPF & RIP Routing
- Configured OSPF and RIP routing protocols across multiple routers
- Verified neighbor adjacency and routing tables
- File: `.pkt` simulation included

### 2. VLAN & Inter-VLAN Routing
- Configured VLANs across multiple switches
- Implemented Inter-VLAN routing using Router-on-a-Stick
- CLI screenshots included

## Tools Used
- Cisco Packet Tracer
- Cisco CLI
